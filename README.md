### 🚀 ルール
- **目的**: 惑星や衛星に衝突せずに、金色のゴールに到達すること。
- **操作**: タップ/クリック。移動ごとにエネルギーを10消費。
- **死神モード**: エネルギーが0になると、最も近い敵が摩擦無視の50倍速で急接近します。
